Dice Roller Version 1

This version of the dice roller is partially procedural like the python version. It is true there is now a control (a button) which when clicked (an 'onClick' event trigger) which calls a function (and event handler), but the process uses a prompt (pop up box) to ask for the input and an alter (pop up box) to display the result. In the next version we will make better use of html form elements to make the app much more event driven and user friendly
